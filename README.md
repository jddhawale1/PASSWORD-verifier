The Cyber Password Strength Analyzer and Wordlist Generator is a web-based cybersecurity tool designed to evaluate password strength and generate customized wordlists. The application provides user registration and login functionality, password strength scoring using zxcvbn, entropy calculation for mathematical security measurement, and generation of attack-specific wordlists based on user inputs. The interface includes a professional cybersecurity dashboard with dark and light mode options, animated strength indicators, and downloadable wordlist files.
To install and run the project locally, Python must be installed on the system. After cloning or downloading the project folder, create a virtual environment and install dependencies listed in requirements.txt using pip. The application can then be started using the command python app.py, and it will run on http://127.0.0.1:5000 in a web browser. The project structure consists of the main Flask application file (app.py), a SQLite database file (users.db), a static folder for CSS styling, and a templates folder containing HTML files for login, registration, and dashboard pages.
This project is intended for educational purposes to demonstrate password security evaluation and cybersecurity concepts. For production deployment, additional security controls such as password hashing, HTTPS encryption, secure session management, and server hardening should be implemented.
